What is Server-Side Rendering?

Server-Side Rendering (SSR) is a technique where web pages are rendered on the server instead of in the browser. The server generates the full HTML content and sends it to the client, which can display it immediately without waiting for JavaScript to execute.

  3. Browser downloads JavaScript
  4. JavaScript executes and renders content
  5. User finally sees the page

  1. Browser requests page
  2. Server renders full HTML
  3. User sees content immediately
  4. JavaScript hydrates for interactivity
  5. App becomes fully interactive
